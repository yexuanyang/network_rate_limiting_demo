# 网络下载速度限流 Demo

一个使用 C++ 实现的简单 TCP 文件下载服务，服务端内置令牌桶算法进行速率限制。

## 功能特点

- **令牌桶算法**：平滑限制下载速率
- **实时进度显示**：客户端显示下载进度和速度
- **灵活配置**：支持命令行参数配置端口、速率等

## 构建

```bash
mkdir build && cd build
cmake ..
make
```

## 使用方法

### 1. 启动服务器

```bash
# 默认配置 (端口 8080, 限速 100 KB/s, 当前目录)
./server

# 自定义配置
./server --port 9000 --rate 51200 --root /path/to/files
```

参数说明：
- `--port`：监听端口（默认 8080）
- `--rate`：速率限制，单位 字节/秒（默认 102400 = 100 KB/s）
- `--root`：文件根目录（默认当前目录）

### 2. 下载文件

```bash
# 下载文件
./client --file example.txt

# 指定服务器地址和输出文件名
./client --file example.txt --host 192.168.1.100 --port 9000 --output my_file.txt
```

参数说明：
- `--file`：要下载的文件名（必需）
- `--host`：服务器地址（默认 127.0.0.1）
- `--port`：服务器端口（默认 8080）
- `--output`：输出文件名（默认 downloaded_<文件名>）

## 示例

```bash
# 终端 1: 启动服务器，限速 50 KB/s
./server --rate 51200

# 终端 2: 下载文件
./client --file test.dat
```

## 架构说明

```
┌─────────────┐         ┌─────────────┐
│   Client    │◄───────►│   Server    │
│             │  TCP    │             │
│ 请求文件名  │────────►│ 读取文件    │
│ 接收数据    │◄────────│ 令牌桶限流  │
└─────────────┘         └─────────────┘
```

### 核心组件

- **`include/rate_limiter.h`**：令牌桶限流器实现
- **`src/server.cpp`**：TCP 文件服务器
- **`src/client.cpp`**：TCP 文件客户端

## 限流算法

使用**令牌桶算法**：
1. 桶中以固定速率生成令牌
2. 发送数据需要消耗对应字节数的令牌
3. 令牌不足时等待，从而实现限速
4. 允许一定程度的突发传输（桶容量）
